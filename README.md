# PrismLauncher Cracked Arch Bin


## Installation

Add to `/etc/pacman.conf`:
```ini
[prism-cracked]
SigLevel = Optional TrustAll
Server = https://repo.oryn.my.id
```

Then install:
```bash
sudo pacman -Sy prismlauncher-cracked-bin
```
